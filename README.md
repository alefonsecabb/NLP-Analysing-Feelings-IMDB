# NLP — Sentiment Analysis on IMDB Reviews (PT-BR)

Sentiment analysis on Portuguese movie reviews using the IMDB PT-BR dataset from Kaggle.

## Dataset
- Source: [IMDB PT-BR — Kaggle](https://www.kaggle.com/datasets/luisfredgs/imdb-ptbr)
- Language: Brazilian Portuguese
- Task: Binary sentiment classification (positive / negative)

## Project Structure
- `data/` — raw and processed datasets (not versioned)
- `notebooks/` — exploratory and modeling notebooks
- `models/` — saved model checkpoints
- `reports/` — figures and evaluation results
- `src/` — reusable Python modules

## Models / Techniques (planned)
- Baseline: TF-IDF + Logistic Regression / SVM
- Deep Learning: BERTimbau fine-tuning
